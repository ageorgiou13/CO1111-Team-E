Heuristics

1.Visibility of system status


When one question answered an informational message appears if its answered correctly or not.

2.Match between system and real world


Simple navigation - user friendly.

3.User control and freedom


User must be able to confirm if he/she wants to skip a question, same on mcq questions in case of a missclick.

4.Consistency and Standards


Consistent design same on all pages.

5.Error prevention


Good error messages and restrictions on input fields.

6.Recognition rather than recall


Clear options on questions like mcq to make the user able to select.

7.Flexibility and efficiency of use


Use of shortcuts for example on QR Scanner copy on clipboard feature.

8.Aesthetic and minimalist design


Consistent and simple design with only 3 colors.

9.Help users recognize, diagnose, and recover from errors


Simple to understand error messages.

10.Help and documentation


Help page to help new users how to use the web app.
